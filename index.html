Tipos de Almacenamiento en Azure

Servicios de Almacenamiento

Azure Blob Storage
Almacena datos no estructurados como archivos, imágenes, videos y respaldos. Funciona como una carpeta en la nube para información no secuencial.
Azure Files
Permite compartir archivos de gran tamaño (GB) mediante herramientas que mantienen los archivos siempre actualizados.
Azure Disk Storage
Disco virtualizado en Azure utilizado principalmente para cargar sistemas operativos en máquinas virtuales.
Azure Table Storage
Almacenamiento estructurado tipo clave-valor (JSON).
Ejemplo: Clave: nombre → Valor: Javier
Azure Queue Storage
Sistema de colas para mensajes asíncronos. Se utiliza con Azure Functions para detonar eventos. Funciona como almacenamiento temporal de mensajes.

Niveles de Acceso

Acceso Frecuente (Hot)
Optimizado para datos con acceso constante. Es el más caro en almacenamiento, pero el más barato en acceso.
Acceso Esporádico (Cool)
Para datos con acceso poco frecuente (al menos una vez al mes). Más económico en almacenamiento, pero más caro en acceso.
Acceso Archivo (Archive)
Para datos que casi nunca se usan. Tiene latencias largas (horas) y el costo de recuperación es alto, aunque el almacenamiento es el más barato.
